# МедЭксперт
## Рекомендуется NodeJS V 14.19.0 + Пакетный менеджер Yarn
### [YARN](https://classic.yarnpkg.com/lang/en/docs/install/)
## Start in dev
```bash
#Install modules
yarn install
#Start app
yarn start
```
## Build
```bash
#Install modules
yarn install
#Start app
yarn build
```
## Build prod
```bash
#Install modules
yarn install
#Start app
yarn build-prod
```
